# TOML++

TOML++ is a header-only TOML parser and serializer for C++17 and later. It
provides a convenient API to read configuration files and work with strongly
typed values in C++.

It is a light-weight option for working with TOML configuration files and can
be found on [GitHub](https://github.com/marzer/tomlplusplus).


## What is it?

1. `main.cpp`: A simple example of using TOML++ to parse a TOML configuration
   file and read typed values from it.
1. `config.toml`: Example configuration file consumed by the program.
1. `CMakeLists.txt`: CMake file to build the example. CMake will automatically
   download and include TOML++ as a dependency.
