# Modularity

Code illustrations for chapter 3, Modularity in Stroustrup's "A tour of C++".


# What is it?

1. `IncludeGuards`: example of include guards, and the problems they solve.
1. `NamespaceParticles`: example of defining a namespace.
1. `ParticlesMake`: example of separate compilation using make.
1. `ParticlesCMake`: example of build using CMake.
1. `Stats`: example of separate compilation.
1. `Modules`: example of using C++20 modules.
