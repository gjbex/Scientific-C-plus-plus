# CLI11

CLI11 is a command line parser for C++11 and beyond. It is a single header
library with no dependencies. It supports subcommands, multiple parsing styles,
and more.

It is a light-weight alternative to `boost::program_options`.


## What is it?

1. `random_default.cpp`: A simple example of using CLI11 to parse command line
   arguments.
1. `CMakeLists.txt`: CMake file to build the example.  CMake will automatically
   download and include CLI11 as a dependency.
