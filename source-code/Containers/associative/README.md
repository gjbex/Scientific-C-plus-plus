# Associative

Code illustrations of set-like and map-like containers.

# What is it?

1. `set.cpp`: illustration of using STL `set` class.
1. `dim_nrs.cpp`: use an unordered set to keep track of dimension numbers
   occurring in a file.
1. `distinct_words.cpp`: reads words from standard input and counts the number
   of distinct values using an unordered set.
1. `dim_nr_counts.cpp`: use an unordered map to count occurrences of dimension
   numbers in a file.
1. `word_frequencies.cpp`: reads words from standard input and finds the word
   with the highest frequency using an unordered map.
1. `aggregate_data.cpp`: use a map from `int` to a statistics class to
   aggregate data and compute average, minimum, and maximum.
1. `data/data.txt`: sample input for `dim_nrs.exe` and `dim_nr_counts.exe`.
1. `data/multi_measurement.txt`: sample input for `aggregate_data.exe`.
1. `CMakeLists.txt`: CMake file to build the applications in this directory.
