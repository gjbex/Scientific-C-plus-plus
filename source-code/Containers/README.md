# Containers

Code illustrations for chapter 9, Containers, in Stroustrup's
"A Tour of C++".


## What is it?

1. `sequence`: examples for sequence containers and tuple-like values.
1. `associative`: examples for set-like and map-like containers.
1. `numeric`: examples that combine containers with numeric algorithms and
   array-like storage.
1. `stacks_queues`: examples for container adaptors such as stacks and
   priority queues.
1. `sliding_window`: class template that uses a list to compute averages over a
   sliding window.

Each subdirectory has a standalone `CMakeLists.txt` and can be configured and
built independently.
