# Diffusion

Event-driven diffusion simulation through a medium.

# What is it?

1. `diffusion_system.h`: declarations of `Particle` and `System` classes to
   simulate diffusion through a medium.
1. `diffusion_system.cpp`: definitions of `Particle` and `System` classes to
   simulate diffusion through a medium. Illustrates use of a priority queue.
1. `diffusion_main.cpp`: main function to drive the diffusion simulation.
1. `CMakeLists.txt`: CMake file to build the application.
