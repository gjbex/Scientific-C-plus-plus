# Containers

Code illustrations for chapter 9, Containers, in Stroustrup's
"A Tour of C++".


## What is it?

1. `sequence`: examples for sequence containers and tuple-like values, including
   exercise solutions based on `std::array`, `std::vector`, and `std::tuple`.
1. `associative`: examples for set-like and map-like containers, including
   exercise solutions based on `std::unordered_set` and
   `std::unordered_map`.
1. `numeric`: examples that combine containers with numeric algorithms and
   array-like storage.
1. `stacks_queues`: examples for container adaptors such as stacks, queues, and
   priority queues.
1. `sliding_window`: class template that uses a list to compute averages over a
   sliding window.
1. `container_choices.md`: short written solution for the container selection
   exercise.

Each subdirectory has a standalone `CMakeLists.txt` and can be configured and
built independently.
