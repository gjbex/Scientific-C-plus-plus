# Stacks and Queues

Code illustrations of container adaptors such as stacks and priority queues.

# What is it?

1. `rpn_calc.cpp`: implementation of a simple calculator for integer
   arithmetic expressions in reverse Polish notation. Illustrates use of a
   stack, a tuple, tokenization, and simple exceptions.
1. `diffusion`: event-driven diffusion simulation that illustrates use of a
   priority queue.
1. `CMakeLists.txt`: CMake file to build `rpn_calc.exe`.
