# Grammar features

A few illustrations of grammar features added in recent C++ standards.


## What is it?

1. `if_init.cpp`: variable defition in `if`-statment (C++17).
1. `structured_bindings.cpp`: illustrates structured binding of a `struct` to
   individual variables (C++17).
1. `struct_initialiation.cpp`: illustrates initializing a `struct` with an
   initializer list (C++20).
1. `CMakeLists.txt`: CMake build configuration for the applications.
