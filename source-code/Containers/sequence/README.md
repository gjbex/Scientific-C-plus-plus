# Sequence

Code illustrations of sequence containers and tuple-like values.

# What is it?

1. `vector.cpp`: illustration of using STL `vector` class.
1. `vector_stats.cpp`: reads floating point values into a `std::vector`,
   then computes minimum, maximum, average, and median.
1. `list.cpp`: illustration of using STL `list` class.
1. `tuple.cpp`: illustration of using STL `tuple` class.
1. `particle_tuple.cpp`: represents particles as tuples and computes the
   Euclidean distance between them.
1. `array_sum_squares.cpp`: reads exactly 10 values into a `std::array`,
   squares them, and computes their sum.
1. `split_string.cpp`: use a vector to store parts of a split string, and
   illustrate `std::string::find()`.
1. `CMakeLists.txt`: CMake file to build the applications in this directory.
