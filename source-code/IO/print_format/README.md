# Print and format

C++20 added a new feature called `std::format` which allows you to format
strings in a more convenient way. It is similar to Python's `str.format`.

C++23 added a new feature called `std::print` which allows you to print formatted
strings directly to the console.


## What is it?

1. `print_format.cpp`: application that demonstrates basic usage of
   `std::format` and `std::print`.
1. `CMakeLists.txt`: CMake file to build the applications.
